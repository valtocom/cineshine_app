# Cineshine — Социальная сеть для киноманов

Выпускная квалификационная работа магистра по направлению подготовки 09.04.02 «Информационные системы и технологии» (профиль: «Веб-разработчик»).

Проект представляет собой MVP (минимально жизнеспособный продукт) веб-платформы, совмещающей функции персонального трекера фильмов и социальной сети с рекомендательной системой.

## Описание проекта

Cineshine — это веб-приложение, разработанное для автоматизации учета просмотренного контента и обеспечения социального взаимодействия между пользователями. Платформа анализирует предпочтения пользователя и его социального графа (друзей) для формирования персонализированной ленты рекомендаций.

Ключевая научная и инженерная ценность: реализация и исследование эффективных алгоритмов фильтрации данных для повышения точности рекомендаций в условиях холодного старта и разреженности матриц оценок.

## Стек технологий

Разработка выполнена в рамках клиент-серверной архитектуры (SPA) с разделением на независимые слои (Frontend / Backend).

| Компонент | Технологии и библиотеки |
| :--- | :--- |
| Frontend | React, TypeScript, Vite, React Router v6, Axios |
| Backend | Node.js, Express, TypeScript |
| База данных | SQLite (реляционная СУБД) |
| Безопасность | JWT (JSON Web Tokens) для авторизации сессий, bcryptjs для асимметричного хеширования паролей с солью |

## Реализованная функциональность

1. Безопасность и профилирование:
   - Регистрация и аутентификация пользователей на базе защищенных JWT-токенов.
   - Персональный профиль пользователя с возможностью редактирования биографии и загрузки аватара.

2. Взаимодействие с контентом:
   - Интерактивный каталог из 200 кинематографических произведений с постерами.
   - Модуль рейтингов: система оценки фильмов по шкале от 1 до 5.
   - Персональный список "Буду смотреть".

3. Социальный граф:
   - Система управления связями: отправка, принятие и отклонение заявок в друзья.
   - Динамическая лента активности, отображающая оценки и действия друзей в реальном времени.

4. Рекомендательная система:
   - Контентная фильтрация: подбор фильмов на основе совпадения жанровых предпочтений.
   - Коллаборативная фильтрация: анализ схожести профилей пользователей.
   - Гибридный алгоритм: продвинутый метод ранжирования, учитывающий веса оценок друзей и жанровое соответствие для минимизации проблемы «холодного старта».

5. Адаптивный интерфейс:
   - Поддержка мобильных устройств, планшетов и настольных компьютеров.
   - Адаптивная навигационная панель с бургер-меню.

## Структура репозитория

Проект организован по принципу монорепозитория с разделением логики клиента и сервера:

```
cineshine_app/
├── backend/               # Серверное приложение (API)
│   ├── src/
│   │   ├── controllers/   # Обработчики запросов и бизнес-логика
│   │   ├── middleware/    # Промежуточное ПО (проверка JWT, валидация)
│   │   ├── routes/        # Эндпоинты API
│   │   └── index.ts       # Точка входа бэкенда
│   └── package.json
├── frontend/              # Клиентское SPA-приложение
│   ├── src/
│   │   ├── components/    # Переиспользуемые UI-компоненты
│   │   ├── pages/         # Страницы приложения (Авторизация, Лента, Профиль)
│   │   ├── App.tsx        # Корневой компонент и роутинг
│   │   └── main.tsx       # Точка монтирования React (Vite)
│   └── package.json
└── README.md
```

## Инструкция по локальному развертыванию

### Системные требования

- Node.js: версия 20.19+ или 22.12+
- npm: версия 9+

### Шаг 1. Запуск серверной части (Backend)

Откройте терминал и выполните команды из корневой папки проекта:

```bash
cd backend
npm install
npm run dev
```

Сервер успешно запустится в режиме автоперезапуска на: `http://localhost:5000`

### Шаг 2. Запуск клиентской части (Frontend)

Откройте второе окно терминала из корневой папки проекта:

```bash
cd frontend
npm install
npm run dev
```

Клиентский сервер разработки Vite запустится на: `http://localhost:5173`

### Шаг 3. Использование приложения

1. Откройте `http://localhost:5173` в браузере.
2. Зарегистрируйте нового пользователя или войдите в существующий аккаунт.
3. Начните оценивать фильмы, добавлять их в список "Буду смотреть" и находить друзей.

## Особенности реализации

1. Все постеры фильмов загружаются с CDN Кинопоиска и доступны без ограничений.
2. Рекомендательная система обновляется в реальном времени при изменении оценок пользователя.
3. Состояние авторизации сохраняется в localStorage, что позволяет не вводить пароль при каждом обновлении страницы.
4. Все запросы к API защищены JWT-токенами, пароли хранятся в захэшированном виде с использованием bcrypt.
