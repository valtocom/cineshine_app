# Cineshine — социальная сеть для киноманов

Выпускная квалификационная работа магистра по направлению 09.04.02 «Информационные системы и технологии» (Веб-разработчик).

## Стек технологий

- **Backend**: Node.js, Express, TypeScript, SQLite
- **Frontend**: React, TypeScript, React Router, Axios
- **Авторизация**: JWT, bcryptjs

## Функциональность

- Регистрация и авторизация пользователей
- Профиль пользователя
- Каталог фильмов (10 тестовых фильмов)
- Оценка фильмов (от 1 до 5)
- Рекомендательная система (коллаборативная фильтрация)

### Backend
