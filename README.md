# Cineshine - социальная сеть для киноманов

Выпускная квалификационная работа магистра по направлению 09.04.02 «Информационные системы и технологии» (Веб-разработчик).


## 📌 Описание проекта

Cineshine - веб-приложение, сочетающее личный дневник фильмов и социальную сеть. Пользователи могут оценивать фильмы, получать персонализированные рекомендации, добавлять друзей и следить за их активностью.


## 🧰 Стек технологий

| Компонент | Технологии |
|---|---|
| **Frontend** | React, TypeScript, React Router, Axios |
| **Backend** | Node.js, Express, TypeScript |
| **База данных** | SQLite |
| **Авторизация** | JWT, bcryptjs |


## Реализованная функциональность
- Регистрация и авторизация (JWT)
- Профиль пользователя
- Каталог фильмов (10 фильмов)
- Оценка фильмов (1–5)
- Рекомендации (коллаборативная фильтрация)
- Добавление в друзья, заявки
- Список друзей
- Лента активности друзей
- Гибридные рекомендации (друзья + жанры)


## Запуск проекта

### Требования
- Node.js 18+
- npm 9+

### 1. Backend
cd backend
npm install
npm run dev
Сервер: http://localhost:5000

### 2. Frontend
cd frontend
npm install
npm start
Приложение: http://localhost:3000
